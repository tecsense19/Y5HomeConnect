# Y5Home Connect

A React application built with Vite and Tailwind CSS.

## Getting Started

### Prerequisites

- Node.js (v18 or newer)
- npm or bun

### Installation

```bash
npm install
```

### Development

To start the development server:

```bash
npm run dev
```

### Production Build

To create a static production build (which outputs to the `dist` folder):

```bash
npm run build
```

## Features

- React 18
- Vite
- Tailwind CSS
- Single Page Application (SPA) routing via TanStack Router
